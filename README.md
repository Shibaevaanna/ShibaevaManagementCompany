# Управляющая компания
## О проекте

**ManagementCompany** — это информационная система для автоматизации работы управляющей компании. Проект предназначен для учета объектов недвижимости, жильцов, коммунальных платежей, показаний счетчиков и формирования квитанций.

## Основные возможности

- Управление лицевыми счетами жильцов.
- Учет квартир и нежилых помещений.
- Ввод и хранение показаний индивидуальных приборов учета (ИПУ).
- Расчет коммунальных платежей по нормативам и по счетчикам.
- Формирование квитанций и отчетов.
- Ведение базы данных на Microsoft SQL Server.

## Технологии

- **Язык** C# 
- **Платформа:** .NET
- **СУБД:** Microsoft SQL Server (разработанный проект подлючен к базе данных)
- **Среда разработки:** Visual Studio 2022
